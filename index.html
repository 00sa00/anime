JS animation toolbox.

Anime.js helps you create smooth JavaScript based animations with a simple, yet powerfull API. Advanced staggerring, easy easings, perfect animations synchronisation, and total control on timings and callbacks. It works with CSS, Canvas, JavaScript objects, SVG or any other DOM attributes.
